# 🪨 Rock Paper Scissors Game ✂️

Welcome to my **first ever web development project** — a simple implementation of the classic **Rock, Paper, Scissors** game using **HTML**, **CSS**, and **JavaScript**. This project was built to learn the basics of frontend development and JavaScript interactivity.

## 🎮 How to Play

- Choose between Rock, Paper, or Scissors by clicking the buttons.
- The computer will randomly select its move.
- The result (Win, Lose, or Draw) will be displayed instantly.
- Keep playing and track your performance!

## 🚀 Live Demo

👉 [Play the game now](https://rock-paper-scissors-game-chi-six.vercel.app)

## 🛠️ Tech Stack

- **HTML** – Structure of the page
- **CSS** – Styling the layout and buttons
- **JavaScript** – Logic and interactivity


## 📚 What I Learned

- Creating and linking HTML, CSS, and JS files
- Handling user input via button clicks
- Using `Math.random()` to simulate computer choice
- Comparing user vs computer moves to determine the result
- Updating the DOM dynamically
- Styling elements with CSS for a clean UI
